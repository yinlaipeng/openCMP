# openCMP IAM 模块开发总结报告

## 项目概述

openCMP (开放多云管理平台) 是一个基于 Go + Gin + Gorm 的单体架构多云管理平台，通过统一接口 + 适配器模式，实现对多个云厂商资源的统一管理和快速接入。

本次开发重点完成了 IAM (身份与访问管理) 模块，包括域、项目、用户、组、角色、权限和策略管理功能。

## 技术架构

### 后端架构
- **分层架构**: Model-Service-Handler-Middleware
- **数据库**: GORM + PostgreSQL/MySQL
- **认证**: JWT Token + bcrypt密码加密
- **安全**: 参数化查询、输入验证、权限控制

### 前端架构
- **框架**: Vue 3 + TypeScript
- **UI组件**: Element Plus
- **状态管理**: Pinia
- **HTTP客户端**: Axios
- **构建工具**: Vite
