# openCMP 用户手册

## 1. 系统概述

openCMP 是一个开源多云管理平台，通过统一的界面管理多个云厂商的资源。

### 1.1 支持的云厂商

- 阿里云（主要支持）
- 腾讯云
- AWS
- Azure

### 1.2 核心功能

- **云账户管理**: 统一管理多云账号
- **资源同步**: 自动同步云资源到平台
- **资源管理**: 虚拟机、网络、存储、数据库等
- **IAM身份管理**: 域、项目、用户、角色、权限
- **消息中心**: 系统通知、消息订阅
- **费用中心**: 账单、成本分析、预算
- **监控告警**: 资源监控、告警策略

## 2. 登录与账户

### 2.1 登录系统

1. 打开浏览器访问系统地址
2. 输入用户名和密码
3. 点击"登录"按钮

默认管理员账户：
- 用户名: `admin`
- 密码: `admin123`

### 2.2 修改密码

1. 点击右上角用户头像
2. 选择"个人设置"
3. 点击"修改密码"
4. 输入当前密码和新密码
5. 点击"确认"

## 3. 云账户管理

### 3.1 添加云账户

1. 进入"云管理 > 云账户"
2. 点击"添加账户"
3. 选择云厂商类型
4. 填写账户信息：
   - 账户名称
   - Access Key ID
   - Access Key Secret
5. 点击"验证凭证"测试连接
6. 点击"保存"

### 3.2 同步资源

1. 在云账户列表中选择账户
2. 点击"同步"按钮
3. 选择要同步的资源类型
4. 等待同步完成

### 3.3 配置同步策略

1. 进入"云管理 > 同步策略"
2. 点击"创建策略"
3. 配置策略内容：
   - 选择云账户
   - 设置同步范围（资源类型）
   - 设置同步规则（项目归属）
4. 保存策略

## 4. 资源管理

### 4.1 虚拟机管理

#### 查看虚拟机列表

1. 进入"计算 > 虚拟机"
2. 选择云账户筛选
3. 查看虚拟机列表

#### 创建虚拟机

1. 点击"创建虚拟机"
2. 选择云账户和区域
3. 配置虚拟机参数：
   - 名称
   - 规格（CPU/内存）
   - 镜像（操作系统）
   - 网络配置（VPC/子网）
   - 安全组
   - 登录密码
4. 点击"确认创建"

#### 虚拟机操作

- **启动**: 点击虚拟机行 → "启动"
- **停止**: 点击虚拟机行 → "停止"
- **重启**: 点击虚拟机行 → "重启"
- **删除**: 点击虚拟机行 → "删除"

### 4.2 网络管理

#### VPC 管理

1. 进入"网络 > VPC"
2. 可创建、查看、删除VPC

#### 子网管理

1. 进入"网络 > 子网"
2. 在VPC下创建子网

#### 弹性IP管理

1. 进入"网络 > 弹性IP"
2. 申请、绑定、解绑EIP

### 4.3 存储管理

#### 云硬盘

1. 进入"存储 > 云硬盘"
2. 创建、挂载、卸载、扩容硬盘

### 4.4 数据库管理

#### RDS 实例

1. 进入"数据库 > RDS"
2. 创建、管理RDS实例

#### Redis 实例

1. 进入"数据库 > Redis"
2. 创建、管理Redis实例

## 5. IAM 身份管理

### 5.1 域管理

域是租户隔离的顶层单位。

1. 进入"IAM > 域管理"
2. 创建、编辑、删除域

### 5.2 项目管理

项目是域内的资源分组。

1. 进入"IAM > 项目管理"
2. 创建项目
3. 添加项目成员
4. 配置项目资源配额

### 5.3 用户管理

1. 进入"IAM > 用户管理"
2. 创建用户
3. 分配用户角色
4. 加入用户组

### 5.4 角色与权限

1. 进入"IAM > 角色管理"
2. 创建角色
3. 为角色分配权限

权限格式：`模块:资源:操作`

示例：
- `compute:vm:list` - 查看虚拟机列表
- `compute:vm:create` - 创建虚拟机
- `network:vpc:create` - 创建VPC

## 6. 消息中心

### 6.1 查看消息

1. 进入"消息中心 > 站内信"
2. 查看系统通知

### 6.2 配置通知渠道

1. 进入"消息中心 > 通知渠道"
2. 添加通知渠道：
   - 邮件
   - 企业微信
   - 钉钉
   - Webhook

### 6.3 配置机器人

1. 进入"消息中心 > 机器人"
2. 添加企业微信/钉钉机器人
3. 配置Webhook地址

### 6.4 消息订阅

1. 进入"消息中心 > 消息订阅"
2. 选择关注的事件类型
3. 配置通知渠道

## 7. 监控告警

### 7.1 配置告警策略

1. 进入"监控 > 告警策略"
2. 创建告警策略：
   - 选择监控资源
   - 设置告警条件（指标、阈值）
   - 设置通知渠道
3. 启用策略

### 7.2 查看告警历史

1. 进入"监控 > 告警历史"
2. 查看历史告警记录
3. 处理告警（确认/忽略）

### 7.3 资源监控

1. 进入"监控 > 资源监控"
2. 选择虚拟机
3. 查看监控指标：
   - CPU使用率
   - 内存使用率
   - 网络流量
   - 磁盘IO

## 8. 费用中心

### 8.1 查看账单

1. 进入"费用中心 > 账单"
2. 按云账户查看账单明细
3. 导出账单数据

### 8.2 成本分析

1. 进入"费用中心 > 成本分析"
2. 查看成本趋势
3. 按项目/服务/资源分析成本

### 8.3 预算管理

1. 进入"费用中心 > 预算"
2. 创建预算：
   - 设置预算金额
   - 设置预算周期
   - 配置预警阈值

## 9. 项目模式切换

系统支持两种模式：

### 9.1 管理控制台模式

显示所有资源和功能，适合全局管理。

### 9.2 项目模式

只显示当前项目相关资源，适合项目成员使用。

切换方式：
1. 点击顶部导航栏的"项目切换"
2. 选择目标项目
3. 界面自动切换为项目视图

## 10. 常见问题

### Q: 云账户验证失败？

检查：
- Access Key 是否正确
- Access Key 是否有相应权限
- 云厂商API是否可访问

### Q: 资源同步失败？

检查：
- 云账户状态是否正常
- 同步策略是否配置正确
- 查看同步日志排查问题

### Q: 权限不足？

联系管理员：
- 检查用户角色
- 检查角色权限
- 检查项目成员配置

### Q: 无法创建资源？

检查：
- 是否有创建权限
- 云账户配额是否充足
- 参数配置是否正确
